## 🎯 核心价值

### 对琪琪（三年级，基础薄弱）

**诊断**：
```
三年级数学，实际水平2.3年级
根本原因：二年级乘法口诀不熟
```

**推荐**：
```
方案1：基础补习 - 二年级乘法口诀（15题，20分钟）
方案2：薄弱突破 - 三年级应用题
方案3：全面复习
方案4：快速练习
```

**效果**：
```
第1周：补习二年级乘法 → 口诀从不会到熟练
第2-4周：三年级内容 → 应用题0%→60%
2个月：达到三年级水平
```
